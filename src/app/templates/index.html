XLSX to PDF Converter

Welcome! This app allows you to upload your Excel spreadsheets (".XLSX" files) into neatly-formatted ".PDF" files. Here are steps to get started:

  • Select a file candidate for upload in the window directly to the right
  • Select a supported file tyle ("eRegulatory", "Financial", "Events")
  • Select "Submit"
  • The file is then converted on the server to .PDF and saved within a database.
  • View all uploaded files below by selecting "Download File" to fetch your corresponding .PDF file from the database.
